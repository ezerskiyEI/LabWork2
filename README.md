 Основные изменения во 2 лабораторной работе:
 Создал и подключил БД PostgreSQL
 Создал таблицы в БД для хранения информации об автомобилях и владельцах
 Реализовал связь @OneToMany между сущностями Owner (владелец) и CarInfo (автомобиль)
 Добавил CRUD-операции для работы с автомобилями и владельцами
 Обновил сервис анализа текста: теперь VIN-номер ищется в БД
 Для проверки использую Postman

 Изменения в классах:
 Добавил новую сущность Owner (владелец автомобиля)
 Добавил репозиторий CarInfoRepository, который позволяет находить машину по VIN
 Метод анализа текста теперь ищет VIN в базе данных, а не просто проверяет текст
 Удалил лишние классы и сервисы, которые дублировали функциональность
